# 🎰 Slot Machine - Python

Bu loyiha Python’da yozilgan oddiy **slot machine (o‘yin avtomati)** dasturidir.

## 🧩 Imkoniyatlar:

* 🎲 Tasodifiy symbol chiqishi (random)
* 💰 Pul tikish (bet qo‘yish)
* 🏆 Yutish yoki yutqazishni aniqlash
* 📊 Balansni kuzatish

## 🚀 Qanday ishlatish:

1. Loyihani yuklab oling:

   ```bash
   git clone https://github.com/bexruzbekgofurboyev-lgtm/Slot_machine.git
   ```

2. Papkaga kiring:

   ```bash
   cd Slot_machine
   ```

3. Dasturni ishga tushiring:

   ```bash
   python main.py
   ```

## 🎮 Qanday o‘ynaladi:

* O‘yinchi pul tikadi
* Slot aylantiriladi
* Agar bir xil symbol tushsa → yutadi 🎉
* Aks holda → yutqazadi ❌
